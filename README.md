## gulp4

- Собрать сборку   
`gulp build`
  

- Запустить   
`gulp`
  

- Оптимизировать изображения  
`gulp images`  
  

- Удалить все изображения   
`gulp cleanimg`
  

- Генерировать сетку SmartGrid Дмитрия Лаврика   
`gulp grid`
  

- Генерировать Критические и Ассинхронные стили  
`gulp criticalgenerate`
